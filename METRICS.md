# Metrics

This repo tracks adoption and quality by a small set of measurable signals.

## Adoption Targets
- Long-term installs target: 50,000+

## What To Track
- GitHub: stars, forks, watchers, issues opened/closed
- Community: PRs merged, unique contributors, discussion threads
- Skill quality: number of actionable outputs per skill; clarity of "inputs" sections

## How To Measure Installs
If using skills.sh or an installer that reports downloads, record those numbers here periodically.
